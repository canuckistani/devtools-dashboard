What is this?

This graph uses the series of 'per user flag' ( for example 'DEVTOOLS_WEBCONSOLE_OPENED_PER_USER_FLAG' ) Telemetry measures to show us relative usage across the tools for specific channel/version combinations, in this case current Beta, Aurora and Nightly versions.

The 'per user flag' measure group sets a preference for a given channel & version combination indicating if the tool has been used at all. This is the best way of indicating rough user population size, based on some assumptions. The potential flaw here is that it may be common for users to have both Aurora and Nightly installed, and so any usage of a given tool in both builds would count that user twice for this measure.

Based on Mozilla's Telemetry open data service.
