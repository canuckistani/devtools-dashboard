What the?

This graph tracks toolbox usage over time comparing the beta, aurora and nightly channels. In particular each point is showing the number of sessions reported where the toolbox is open for more than 5 minutes. Note in particular that telemtry collection is now opted-in by default for Beta starting in early/mid June, an that there is a corresponding huge jump in the number of sessions collected there.

Based on Mozilla's Telemetry open data service.
